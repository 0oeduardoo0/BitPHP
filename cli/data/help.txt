+-----------------------------------------+
|  BitPHP Command Line Interface v 1.0.0  |
+-----------------------------------------+

 usage: php dummy [action] [parameters]

 actions:
   init   - Create new app
         
            [optional parameters]
            mvc       - To create a new mvc application
            hmvc      - To create a new hmvc application
            micro     - To create a new micro mvc app
            rest      - To create a new RESTful app

   create - Create something
            
            [paremeters]
            config.json       
                              - Create new config file in /app
            config.json <dirpath> 
                              - Create new config file in <dir>

   error - Cheking for errors

            [parameters]
            ERROR_ID
                              - Check info of specific error
            dump
                              - Clear error log
            all
                              - Check for all errors



   update - Check for updates

   remove - Delete de app

   restore - Restore the last removed app


[EXAMPLES]

   php dummy init 
   - To create a blank app

   php dummy init mvc 
   - Initialize a MVC service

   php dummy create config.json 
   - To create a config.json in app/

   php dummy create config.json /home/User/Documents/
   - To create a config.json in /home/User/Documents/

   php dummy revome
   - To remove the app
